Command line Help

  1. JDK download and installation:
  2. download from Oracle site
  3. setup: JAVA_HOME environment variable
  4. add in PATH environment variable
  5. create Git account.
  6. create Fork from: https://github.com/rubfan/hillel-java-elementary
  7. create Slack account.
  8. copy link to your fork.
  9. enter to your command line: ⇒ git clone your_fork_link
  10. open created folder and go to the current "task_0" folder
  11. create your own folder: firstname.lastname.rownumber.placenumber
  12. go to this folder: ⇒ cd firstname.lastname.rownumber.placenumber
  13. create java file: ⇒ touch YourClass.java
  14. edit your file: ⇒ vi YourClass.java
  15. write necessary code and accomplish your current task.
  16. compile your java file: ⇒ javac src/YourClass.java -d out
  17. start compiled class: ⇒ java -cp out YourClass
  18. create java docs: ⇒ javadoc -d docs src/YourClass.java
  19. ⇒ git status
  20. ⇒ git add .
  21. ⇒ git status
  22. commit your changes: ⇒ git commit -m "some comments about your changes"
  23. push your changes to yor fork repository: ⇒ push origin master
  24. go to your git fork page and check all changes.
  25. click the button "Pull Request" and create PR form your fork branch to master.
  26. https://help.github.com/articles/configuring-a-remote-for-a-fork/
  27. ⇒ git remote -v
  28. ⇒ git remote add upstream https://github.com/user/repo.git
  29. ⇒ git remote -v
  30. ⇒ git pull upstream master
  31. ⇒ git push origin master
